# /draft - Write Full PRD Draft

## Purpose

Write a comprehensive PRD draft based on the synthesized outline, discovery transcript, and PRD template.

## Prerequisites

- Discovery transcript: `artifacts/prd-creation/discovery.md`
- Synthesized outline: `artifacts/prd-creation/outline.md`
- PRD template: `templates/prd-template.md`

## Process

1. **Read all inputs**

   Read the following files:
   - `artifacts/prd-creation/discovery.md`
   - `artifacts/prd-creation/outline.md`
   - `templates/prd-template.md`

2. **Write the full PRD draft**

   Act as a senior Product Manager writing a comprehensive PRD draft.

   Write a complete PRD draft that:
   - Follows the template structure for all included sections
   - Fills in concrete details based on the discovery conversation and specialist inputs
   - Uses clear, precise language appropriate for a technical audience
   - Includes specific, measurable criteria where possible
   - Marks any areas of uncertainty with [TBD] and a note about what needs to be resolved
   - Replaces all template placeholder text with actual content
   - Maintains professional formatting with proper markdown

   Do NOT include template instruction text (the guidance paragraphs). Replace them entirely with actual content.

3. **Write the draft**

   Write the complete PRD draft to `artifacts/prd-creation/draft.md`.

## Output

- `artifacts/prd-creation/draft.md` — Complete PRD draft

## Next Steps

After drafting, run `/review` to launch specialist reviews.
